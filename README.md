# Algoritmos 

## 📖 **Sobre**
É um repositório com algoritmos, para quem está iniciando.

###  Como funciona
Primeiramente existe um arquivo do tipo ".md" com a ideia do algoritmo. Depois um arquivo do tipo ".py" com a ideia, algoritmo, em python.  
Exemplo: 
~~~
1_1_começo.md  
1_2_começo.py
~~~
